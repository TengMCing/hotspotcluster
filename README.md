Perform spatiotemporal clustering on this dataset.

``` r
results <- hotspot_cluster(hotspots5000, 
                           lon = "lon", 
                           lat = "lat", 
                           obstime = "obstime",
                           ActiveTime = 24,
                           AdjDist = 3000,
                           ignition_center = "mean",
                           time_unit = "hours",
                           timestep = 1)
#> Transform time_id <U+2713> 
#> Clustering <U+2713> 
#> Compute ignition points <U+2713> 
#> Number of clusters: 217

```
